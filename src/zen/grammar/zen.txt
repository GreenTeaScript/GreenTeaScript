Zen 仕様書

Zen パーサー用ライブラリの言語仕様

シンボル定義
============

let a = 1;

関数定義
========

型なし
function f(a, b) {
	return a + b;
}

型あり
function f(a: int, b: int): void {
	return a + b;
}

変数宣言
========

型なし
var x = 1;

型あり
var x: int = 1;

初期値は常に必要である

}

}
